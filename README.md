# Pipeline for Optical Mapping — Review & Progress Tracking

Репозиторий для систематического ревью и рефакторинга пайплайна обработки данных оптического картирования сердца (cardiac optical mapping).

## Быстрый старт

### Для понимания текущего состояния

1. Начните с **`tracking/PROGRESS_TRACKER.md`** — общая картина прогресса
2. Для деталей по конкретному фиксу → **`tracking/FIX_TRACKER.md`**
3. Для статуса модуля → **`tracking/MODULE_STATUS.md`**

### Для продолжения ревью

1. Прочитайте последние записи в **`tracking/REVIEW_LOG.md`**
2. Используйте чек-листы из **`docs/review/REVIEW_GUIDE.md`**
3. После сессии — добавьте запись в REVIEW_LOG (append-only)

### Для понимания архитектуры

1. **`docs/architecture/PIPELINE_STRUCTURE.md`** — текущие 8 стадий + предлагаемые агенты
2. **`docs/architecture/Pipeline_Organization_Proposal.md`** — целевая архитектура

## Как вносить изменения

1. **Фиксы:** Обновляйте статус в `tracking/FIX_TRACKER.md`
2. **Ревью:** Добавляйте записи в `tracking/REVIEW_LOG.md` (только append)
3. **Новые баги:** Добавляйте в конец соответствующей секции FIX_TRACKER с новым ID
4. **Прогресс:** Пересчитывайте сводку в `tracking/PROGRESS_TRACKER.md`

## Контекст проекта

Пайплайн обрабатывает данные оптического картирования сердца:
- **Вход:** `.gsd` файлы (видео флуоресценции)
- **Стадии:** Загрузка → Маскирование → Активация → APD → CV → Альтернанс → OWS → Фенотип
- **Выход:** Карты APD, CV, альтернанса + финальный фенотип

Поддерживаются два красителя:
- **Dye A (VSD):** Voltage-sensitive dye → APD, activation
- **Dye B (Ca²⁺):** Calcium indicator → CaT, rise/decay time
